# User Spec

The point of this code base is to facilitate the AI powered planning process on a VPS. I have Claude Code, Codex and other agentic coding CLIs installed on my virtual private server. Because it's a virtual private server that runs in headless mode, there is no desktop environment and I cannot open the markdown implementation plans that Claude or Codex has for me. Because of this, I created this code base. This code base allows Claude or Codex or Gen and I or whoever to give me a URL that I can use to locally open in a browser the markdown implementation plan the AI wrote and I can edit it in that browser as well where it is being rendered with the markdown rendering to be scannable and readable and aesthetically consumable. I'd like you to make three highly targeted, very high value improvement suggestions to this repo in order to facilitate that process. Of course, after you have familiarized yourself with the existing state of the code base.
